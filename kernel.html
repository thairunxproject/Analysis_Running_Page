Kernel สำหรับการจำลองข้อมูล

: สูตรในการจำลองข้อมูล

: แบ่งกลุ่ม Properties เป็น 4 ส่วนเพื่อให้ใช้สำหรับการจำลองได้แก่

: กลุ่มแรก (แกน X ) = คือกลุ่มของคะแนนส่วนของ Promote, Review, Advertise

: กลุ่มสอง (แกน Y ) = คือกลุ่มของคะแนนส่วนของ Quotes & Interviews, Entertain, Articles

: กลุ่มสาม ( รัศมีของวงกลม ) = คือจำนวนบัญชีที่กดไลก์เพจ

: กลุ่มสี่ ( สีของวงกลม ) = คือความถี่ในการโพสต์ต่อวัน



กลุ่มแรก และ กลุ่มที่สอง จะต้องนำข้อมูลมาทำการจำลองตำแหน่งบนแกน X และ แกน Y โดนจะนำคะแนนในแต่ละด้านของกลุ่มมารวมกัน และ ลบด้วย (คะแนนกึ่งกลาง)*
เช่น ( P1 + P2 + P3 ) - XX = ตำแหน่งบนแกน X , ( P4 + P5 + P6 ) - XX = ตำแหน่งบนแกน Y

(คะแนนกึ่งกลาง) = การจำลองคะแนนที่อยู่ในระดับพื้นฐาน และ จำนวนโพสต์ที่อยู่ในระดับพื้นฐาน มาคำนวนเพื่อสามารถให้จำลองแบ่งแกนเป็น 4 ส่วนได้
ซึ่งกำหนดให้คะแนนพื้นฐานคือ 5 และ จำนวนโพสต์พื้นฐานคือ 5 ทำให้ผลของคะแนนกึ่งกลางคือ 75 คะแนน (มาจากคะแนนทั้งสองกลุ่มคำนวณกันเพื่อให้ได้คะแนนกลาง)


กลุ่มที่สาม (รัศมีของวงกลม) จะเก็บอยู่ในรูปแบบข้อมูลหน่วยหลัก K (1000 บัญชี) เผื่อให้สามารถแบ่งแยกได้ง่ายขึ้น


กลุ่มที่สี่ (สีของวงกลม) จะไล่ระดับสีตามความถี่ของการโพสต์ในแต่ละวัน
